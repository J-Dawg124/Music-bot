A web app that lets you search albums using the Spotify API, save them to your personal library, and track listening habits with ratings, reviews, and stats.

🚀 Live Demo

👉 https://music-bot-ecru.vercel.app/

✨ Features
🔍 Search albums from Spotify database
➕ Add albums to your personal tracker
🎧 Mark albums as listened / not listened
⭐ Rate and review albums
📊 Stats dashboard:
Total albums
Average rating
Top artists
💾 Saves data locally in your browser (localStorage)
🔐 Spotify login (OAuth PKCE flow)
